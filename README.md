# devops-netology

Репозиторий для домашних заданий и проектов курса DevOps от Netology.

Здесь сохраняются скрипты и конфигурации для различных систем,
изучаемых на протяжении курса.

## Игнорируемые файлы (.gitignore)

В каталоге `terraform/` находится файл `.gitignore`. Ниже расшифровано, файлы
**какого вида** (по их именам) будут игнорироваться каждым правилом.

Напоминание о языке правил:
- `*` — любая последовательность символов (в том числе пустая) в пределах одного
  имени (символ `*` не «перешагивает» через `/`);
- `.` — обычный символ «точка»;
- завершающий `/` означает, что правило относится к каталогу;
- правила записаны без ведущего `/`, поэтому действуют в каталоге `terraform/`
  и в любом его подкаталоге.

Правила и что под них попадает:

- `.terraform/` — каталог с именем `.terraform` и всё его содержимое;
- `*.tfstate` — все файлы, имя которых **оканчивается** на `.tfstate`
  (например `terraform.tfstate`, `prod.tfstate`);
- `*.tfstate.*` — все файлы, в имени которых встречается `.tfstate.` с любым
  текстом до и после (например `terraform.tfstate.backup`, `prod.tfstate.1`);
- `crash.log` — файл с точным именем `crash.log`;
- `crash.*.log` — файлы, имя которых **начинается** на `crash.` и **оканчивается**
  на `.log`, с любым текстом между ними (например `crash.2024-01-01.log`);
- `*.tfvars` — все файлы, оканчивающиеся на `.tfvars`;
- `*.tfvars.json` — все файлы, оканчивающиеся на `.tfvars.json`;
- `override.tf` — файл с точным именем `override.tf`;
- `override.tf.json` — файл с точным именем `override.tf.json`;
- `*_override.tf` — все файлы, имя которых оканчивается на `_override.tf`
  (перед этим — любой текст, например `aws_override.tf`);
- `*_override.tf.json` — все файлы, оканчивающиеся на `_override.tf.json`
  (например `aws_override.tf.json`);
- `.terraform.tfstate.lock.info` — файл с точным именем `.terraform.tfstate.lock.info`;
- `.terraformrc` — файл с точным именем `.terraformrc`;
- `terraform.rc` — файл с точным именем `terraform.rc`.

Корневой файл `.gitignore` зарезервирован для общих правил репозитория.
